== Description ==
Lisäosalla pystyt arpomaan helposti itsellesi lottorivin. Tällä hetkellä Lotto, Viking Lotto ja Eurojackpot on tuettuna. Voit myös hieman varioida näiden vaihtoehtojen välillä. 

Tämä lisäosa sopii hyvin erilaisiin sivustoihin, jotka ovat keskittyneet pelaamiseen. Liitä lisäosa sivupalkkiin tai sivupohjaan eli footeriin ulkoasu -> vimpaimet alta. 

Loton onnennumerot on erittäin helppokäyttöinen lisäosa, joten sivusi vierailijat varmasti osaavat sitä käyttää. 

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Liitä lisäosa sivupalkkiin, footeriin tai pikakoodilla siihen kohtaan sivua mihin sen haluat.
